ByteBrew Widget Example

This page demonstrates the ByteBrew embeddable chat widget.

Setup

  1. Start ByteBrew Engine: docker compose up -d
  2. Create an agent in Admin Dashboard (http://localhost:8443/admin)
  3. Mark the agent as Public
  4. Create an API key with Chat (Public) scope
  5. Replace YOUR_API_KEY and YOUR_AGENT_NAME below
  6. Open this file in a browser

How It Works

The widget is loaded via a single <script> tag. It creates a chat bubble in the bottom-right corner. Click to open, type a message, and get AI responses streamed in real-time.

Configuration

<script src="http://localhost:8443/widget.js"
        data-agent="YOUR_AGENT_NAME"
        data-api-key="YOUR_API_KEY"
        data-theme="light"
        data-position="bottom-right"
        data-title="AI Assistant">
</script>
